# Parcial 1
## Para C
- Los archivos process.c y process_c hacen parte de la implementación en C para ejecutar la implementacion en C use: ./process_c para hacer el proceso por apuntadores con 1 millón de datos generados o puede generar los datos que desee indicando despues de ./process_c el numero de datos que desea:./process_c 10000, para hacerlo por valor use: ./process_c 10000 value indicando el numero de datos a generar y la palabra value  
## Para C++ 
- Del mismo modo use ./process_cpp para generar por defecto 1 millón de datos y procesarlos por apuntadores, de otro modo use ./process_cpp 10000 value para generar datos del numero deseado y procesar por valor
